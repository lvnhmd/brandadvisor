What is Spicy Treats UI Kit

Spicy Treats UI Kit is a number of useful user interface components built on Bootstrap 3.2.0 framework. We did not modify any line of Bootstrap, so you can safely use Spicy Treats UI Kit in your ongoing project.

What is included

Once purchased, unzip the compressed folder to see the directory structure of Spicy Treats UI Kit. You'll see something like this:

Note: You can modify css files directly, but it's better to modify less files and generate css from them.

Browser support

Since Spicy Treats UI Kit is built on Bootstrap 3 framework, it supports the same browsers as Bootstrap.

Getting Started

To start working with Spicy Treats UI Kit you can use a basic template.html, or take the completely functional pages included in this package. Since it is built on Bootstrap, you can also use it on your project with Bootstrap 3.

We provide you LESS version for better customization of your project. We built it Bootstrap way so you won't find big difference. Use less/variables.less to modify different styles and style.less to remove needless components. After all modifications you will need to recompile style.less and new style.css file will be generated with new variables. For compiling less files you can use WinLess but you can use less compiler you like.

After finishing the customization part you'll have to drop all contents of html folder to your project folder and including style.css file in your HTML page: <link href="style.css" media="screen" rel="stylesheet">

If you want to start from scratch, the best way is to use our template where all startup files are already included and ready to be used. You can find template.html file in /html/ folder.

    jQuery(document).ready(function() {
        var graphwidth = $('.widget-graph .widget-inner').width();
        $('.widget-graph .graph').css('height', parseInt(graphwidth/1.2));
        $(window).resize(function() {
            graphwidth = $('.widget-graph .widget-inner').width();
            $('.widget-graph .graph').css('height', parseInt(graphwidth/1.2));
        });

        var d1 = [[0.5, 5],[1, 20], [1.7, 15], [2.5, 25], [3, 18], [4, 36]],
                graphholder = $("#graph"),
                plot = $.plot(graphholder, [d1], {
                    colors: ["#ae5728"],
                    xaxis: {
                        min: 0,
                        max: 4.5
                    },
                    yaxis: {
                        min: 0,
                        max: 40
                    },
                    series: {
                        lines: {
                            show: true,
                            lineWidth: 4,
                            fill: true,
                            fillColor: "rgba(0, 0, 0, 0.15)"
                        },
                        points: {
                            show: true,
                            radius: 6,
                            lineWidth: 4,
                            fillColor: "#d19b7d"
                        },
                        shadowSize: 3
                    },
                    grid: {
                        hoverable: true,
                        clickable: true,
                        margin: 20,
                        color: "#555053",
                        borderColor: "#3e393c"
                    }
                });

        function showTooltip(x, y, contents) {
            $("<div id='graph-tooltip'>" + contents + "</div>").css({top: y - 45, left: x - 22}).appendTo("body").fadeIn(200);
        }

        var previousPoint = null;
        graphholder.on("plothover", function (event, pos, item) {

            if (item) {
                if (previousPoint != item.dataIndex) {

                    previousPoint = item.dataIndex;

                    $("#graph-tooltip").remove();
                    var x = item.datapoint[0].toFixed(2),
                            y = item.datapoint[1].toFixed(2);

                    showTooltip(item.pageX, item.pageY, y*10);
                }
            } else {
                $("#graph-tooltip").remove();
                previousPoint = null;
            }
        });
    });
</script>
